## 💡 设计原则

### 1. 信息层次化

- 使用标题、边框、颜色区分层级
- 重要信息突出显示
- 次要信息适当弱化

### 2. 视觉一致性

- 统一的边框样式
- 一致的间距和对齐
- 规范的编号格式

### 3. 扫描友好

- 表格化呈现便于快速查找
- 关键字高亮
- Emoji 辅助识别

### 4. 响应式考虑

- 表格使用百分比宽度
- 支持 GitHub 渲染
- 移动端友好

### 5. 可读性优先

- 适当的留白
- 清晰的分组
- 简洁的描述
